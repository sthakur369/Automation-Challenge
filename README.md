# Automation-Challenge

This is an test automation script to test UI of this link https://dev.deepthought.education/login

**Test Cases:**

> Invalid Username

> Invalid Password

> Password too short

> Succesful Login

**Necessary Installation:**

* NodeJS: https://nodejs.org/en/download

* TestCafe:   `npm install -g testcafe`


**To run the script:**

`testcafe chrome Automation_script.js`

OR

`testcafe opera Automation_script.js`
